Este frente contiene únicamente casos del piloto en español para el Agente Especializado en Lenguaje Natural para el SV.

## Subordinación

El frente español no decide por sí mismo. Sólo aporta casos tipados para:

- `I_NLP`
- `H_NLP`
- `Γ_H`
- `T_NLP`
- `κ_3`

## Estructura

- `casos/`: casos individuales por familia.
- `lotes/`: consolidaciones iniciales del piloto.
